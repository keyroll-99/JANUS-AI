# Janus AI - Intelligent Investment Portfolio Assistant

![GitHub issues](https://img.shields.io/github/issues/your-username/janus-ai)

Janus is an open-source web application (with a mobile-responsive view) designed to simplify personal investment portfolio management. It allows importing data from XTB, AI-powered portfolio analysis, and customized recommendations.

---

## 🎯 The Problem

Managing an investment portfolio, often scattered across different accounts (main portfolio, IKE, IKZE), is complicated and cumbersome. Investors, especially beginners, struggle to decide when to buy or sell assets and how to stay consistent with their strategy. They lack a simple tool that aggregates data and provides personalized recommendations.

## ✨ Key Features

- **Data Import**: Quickly populate the application by importing transaction history from Excel files generated by XTB.
- **Transaction Management**: Manually add and edit transactions.
- **Strategy Definition**: A simple form to define investment goals, time horizon, and risk level.
- **AI Analysis**: One-click portfolio analysis by Claude/Gemini, generating a summary and "buy/sell" recommendations.
- **Dashboard**: A clear dashboard with total portfolio value, historical chart, and asset diversification visualization.
- **Analysis History**: An archive of all performed analyses to track changes in recommendations over time.

## 🛠️ Tech Stack

### Backend
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Language**: TypeScript
- **Key Libraries**: Passport-JWT (auth), Multer (file upload), Zod (validation), @anthropic-ai/sdk

### Frontend
- **Framework**: React 19
- **Routing**: React Router v7
- **Styling**: SCSS
- **Build Tool**: Vite
- **Key Libraries**: React Hook Form (forms), Recharts (charts)

### DevOps
- **Containerization**: Docker
- **CI/CD**: GitHub Actions

## 🏗️ Project Architecture

The project is organized as a monorepo with a clear separation between backend and frontend.

```
janus-ai/
├── backend/
│   ├── src/
│   │   ├── domains/          # Business Logic (DDD)
│   │   ├── shared/           # Shared Modules
│   │   ├── app.ts            # Main Express App
│   │   └── server.ts         # Server Initialization
│   └── package.json
│
├── frontend/
│   ├── src/
│   │   ├── components/       # UI Components
│   │   ├── pages/            # App Views
│   │   ├── shared/           # Shared Hooks, API, Types
│   │   ├── App.tsx           # Main App Component
│   │   └── main.tsx          # Entry Point
│   └── package.json
│
└── .github/
    └── workflows/            # CI/CD Automation
```

## 🚀 Getting Started

### Prerequisites
- Node.js (version from `.nvmrc` recommended)
- Docker and Docker Compose
- Access to a PostgreSQL instance

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/janus-ai.git
cd janus-ai
```

### 2. Backend Setup
```bash
cd backend

# Copy the .env.example file and fill in your variables
cp .env.example .env

# Install dependencies
npm install

# Start the development server
npm run dev
```
The backend server will be available at `http://localhost:3000`.

### 3. Frontend Setup
```bash
cd ../frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```
The frontend application will be available at `http://localhost:5173`.

## 🎯 Problem

Zarządzanie portfelem inwestycyjnym, często rozproszonym między różne konta (główny portfel, IKE, IKZE), jest skomplikowane i nieporęczne. Inwestorzy, zwłaszcza początkujący, mają trudności z oceną, kiedy sprzedać lub kupić aktywa oraz jak utrzymać spójność ze swoją strategią. Brakuje im prostego narzędzia, które agreguje dane i dostarcza spersonalizowanych rekomendacji.

## ✨ Kluczowe Funkcjonalności

- **Import Danych**: Szybkie zasilanie aplikacji przez import historii transakcji z plików Excel generowanych przez XTB.
- **Zarządzanie Transakcjami**: Możliwość ręcznego dodawania i edytowania transakcji.
- **Definicja Strategii**: Prosty formularz do zdefiniowania celów inwestycyjnych, horyzontu czasowego i poziomu ryzyka.
- **Analiza AI**: Uruchamiana jednym przyciskiem analiza portfela przez Claude/Gemini, generująca podsumowanie i rekomendacje "kup/sprzedaj".
- **Dashboard**: Przejrzysty pulpit z łączną wartością portfela, wykresem historycznym i wizualizacją dywersyfikacji aktywów.
- **Historia Analiz**: Archiwum wszystkich przeprowadzonych analiz do śledzenia zmian w rekomendacjach.
